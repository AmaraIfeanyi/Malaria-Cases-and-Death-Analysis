# 🦟 Malaria Cases and Deaths Analysis Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyse global malaria cases and deaths across different countries and years. The dashboard provides insights into malaria trends, high-risk countries, mortality rates, and yearly comparisons to support data-driven health awareness and decision-making.

The project focuses on transforming raw health data into meaningful visual insights using data analytics and storytelling techniques.

---

## 🎯 Objectives
- Analyse malaria cases and deaths across multiple countries
- Identify countries with the highest malaria burden
- Track yearly trends in malaria cases and mortality
- Calculate and visualise Case Fatality Rate (CFR%)
- Create an interactive dashboard for easy exploration of data

---

## 📊 Dashboard Features
- Total malaria cases overview
- Total malaria deaths overview
- Mean cases and deaths
- Case Fatality Rate (CFR%)
- Yearly trend analysis
- Top countries by malaria cases
- Top countries by malaria deaths
- Interactive filters by country and year

---

## 🛠️ Tools & Technologies Used
- **Power BI** – Data visualisation and dashboard creation
- **Microsoft Excel / CSV** – Data source and cleaning
- **DAX** – Calculated measures and KPIs
- **Power Query** – Data transformation and modelling

---

## 📈 Key Insights
- Nigeria recorded the highest malaria cases and deaths among analysed countries.
- Malaria deaths showed a declining trend over the years analysed.
- Certain African countries continue to carry a significant malaria burden.
- The dashboard helps uncover patterns useful for public health analysis.
